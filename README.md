## コミットリマインダー

GitHub Actions で毎日のコミット習慣を維持するためのリマインド通知を送信する。

- 未コミット時に Pushover 経由で iPhone に通知
- 1日3回、段階的にリマインド（20:00 / 21:00 / 22:00 JST）
- 詳細は [docs/REQUIREMENTS.md](docs/REQUIREMENTS.md) を参照
